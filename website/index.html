About System Operating

System Operating is an educational project that brings operating system concepts to microcontrollers. Built for the ESP8266 and ESP32 platforms, it provides a Unix-like shell environment with file system support, network connectivity, and extensible application modules.

Status: Early Development (Unstable)
Target Hardware: ESP8266 NodeMCU, ESP32
Display: OLED (SSD1306 compatible)
File System: LittleFS

Project Goals

  • Learn operating system fundamentals through hands-on implementation
  • Create a functional shell environment on resource-constrained hardware
  • Implement file system operations (create, read, write, delete)
  • Support modular applications and interpreters
  • Provide network services (WiFi, NTP, weather data)

Core Features

Shell Interface

Interactive command-line interface accessible via serial monitor. Parse and execute commands with argument support.

File System (LittleFS)

Full Unix-like file operations: navigate directories, create/delete files, copy/move files, and edit text files.

Network Stack

WiFi connectivity with status monitoring, network time synchronization (NTP), and external API integration.

System Monitor

Real-time system statistics including heap memory usage, uptime tracking, and hardware information.

Application Modules

  • Brainfuck Interpreter: Run Brainfuck programs directly on the microcontroller
  • Weather Service: Fetch and display current weather conditions via API
  • Time & Date: Network time protocol (NTP) synchronization
  • ELIN Language: Custom stack-based programming language with compiler and VM (in development) - View Documentation

Available Commands

File System Commands

Command Description Example
ls List files and directories ls
cd [path] Change directory cd /home
pwd Print working directory pwd
cat [file] Display file contents cat readme.txt
touch [file] Create new file touch notes.txt
mkdir [dir] Create directory mkdir projects
rm [file] Remove file rm temp.txt
rmdir [dir] Remove directory rmdir old_folder
cp [src] [dst] Copy file cp file.txt backup.txt
mv [src] [dst] Move/rename file mv old.txt new.txt
lexa [file] [content] Write to file lexa test.txt Hello World

System Commands

Command Description
help Display available commands
stats Show system statistics (memory, uptime)
wifi Display WiFi connection status
weather Fetch current weather data
bf Run Brainfuck interpreter

Get Started

Hardware Requirements

  • ESP8266 NodeMCU or ESP32 development board
  • OLED Display (SSD1306, 128x64 recommended)
  • USB cable for programming and serial communication

Software Requirements

  • Arduino IDE or PlatformIO
  • ESP8266/ESP32 board support package
  • Required libraries: LittleFS, Adafruit_SSD1306, WiFi
